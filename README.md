# Getting Started

## Dev environment prerequisites
Node JS install: https://nodejs.org/en/

Git install: https://git-scm.com/

## Dev environment setup
`git clone https://github.com/scbcmd/scbc-app.git`

`cd scbc-app\website`

`npm install`

## Starting The Application
`npm start`
